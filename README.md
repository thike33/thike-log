# Next.jsで自分のブログサイト構築

## 環境
- Nodeバージョン 18.13.0
- Next.jsはApp Router
- TailWindCSS使用
- CMS → MicroCMS

## 使用方法
- dev　→　開発環境起動
- build　→　ビルド
- start　→　本番環境起動
- lint　→　lint:eslintとlint:prettierを並列実行
- lint:eslint　→  ESLintチェック
- lint:prettier　→　prettierチェック
- fix　→　fix:eslintとfix:prettierを並列実行
- fix:eslint　→　ESLintで自動修正
- fix:prettier → prettierで自動整形
